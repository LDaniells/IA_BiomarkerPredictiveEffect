# IA_BiomarkerPredictiveEffect
Code for the paper titled 'Adaptive Trial Designs for Assessing Predictiveness of a Biomarker in Early Phase Drug Development' by Libby Daniells, Julia Geronimi, Hugo Hadjur, Sandrine Guilleminot and Pavel Mozgunov.

Files are as follows: 
- SimulationCode.R runs all simulations within the paper. Interim and total sample sizes can be edited in the specified places, as can the biomarker distribution.
- IllustrativeExample.R recreates the illustrative example in the supplementary materials.
